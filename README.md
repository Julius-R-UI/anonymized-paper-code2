# DINO Fork

**NOTE: This is a fork of DINO (DINOv3) with modifications.**

This repository contains a modified version of the DINO (DINOv3) codebase. The original DINOv3 project was developed by Meta Platforms, Inc. and affiliates.

## Original DINO

This codebase is based on DINOv3, which is available from the original repository.

## Modifications

This fork includes various modifications and extensions to the original DINO codebase. These are the files we as authors of HASSL focused on:
1. /dinov3/train/train.py
2. /dinov3/loss/contrastive_hierarchical_loss.py
3. /dinov3/train/ssl_meta_arch.py
